#🧠 EmotionFlow – AI Powered Study Assistant

EmotionFlow is a smart web application that detects a user's emotional state using AI and provides a personalized study plan in real-time. It helps learners stay productive, avoid burnout, and improve focus through adaptive recommendations.

---

## 🚀 Features

- 🎭 **Real-time Emotion Detection** using AI (Gemini API)
- 📊 **Dynamic Dashboard**
  - Focus Score
  - Mood Tracking
  - Burnout Risk
- 📚 **Personalized Study Plan**
  - Do’s and Avoid’s based on emotion
- ⚡ **Instant Updates**
  - Dashboard + Study Plan update live
- 🔐 **Authentication**
  - Email/Password Login
  - Google Sign-In (Firebase)
- 💾 **MongoDB Integration**
  - Stores user records and history
- 🔄 **Socket.io Integration**
  - Real-time communication between client & server
